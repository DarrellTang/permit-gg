## Decisions Made
- Set capture_pageview: false and use custom PostHogPageView component because App Router misses client-side navigations with auto-tracking

- Removed /practice from PROTECTED_PATHS array so first free quiz can proceed without auth; cookie-based gate handles subsequent attempts
- Used both localStorage and server cookie for free quiz tracking: cookie for middleware enforcement, localStorage for instant client-side UX
