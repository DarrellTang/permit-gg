**Create src/lib/posthog/identify.tsx:** ('use client')

  4. Returns cleanup: subscription.unsubscribe()
- Returns null
- Follow 2.1-RESEARCH.md PostHog User Identification pattern

**Free quiz gating — update src/middleware.ts:**
Add free quiz gate logic to the existing middleware (Plan 01 created this file):
- If user is NOT authenticated AND path is /practice:
  - Check for `permit_free_quiz_used` cookie
  - If cookie exists: redirect to /login with ?next=/practice
  - If cookie does NOT exist: allow through (first free quiz)
- Do NOT gate /simulated-test for free quiz — sim test always requires auth (per RESEARCH.md open question #2: "one free practice quiz" limits free quiz to /practice only)
- The cookie will be set by the quiz completion flow (see use-quiz.ts changes below)

**Free quiz cookie — update src/hooks/use-quiz.ts:**
- Import analytics from @/lib/posthog/events
- In startQuiz: call analytics.quizStarted(quizMode, questionCount) (or analytics.simTestStarted() if sim mode)
- In handleSubmit (or after each answer via a new effect): call analytics.answerSubmitted(isCorrect, categorySlug, streak) — this needs the current question's category slug. Add a `categorySlug` param or read from currentQuestion in the store.
- In handleComplete: after saveQuizResults, call analytics.quizCompleted(mode, score, scorePercentage, totalTimeMs). If sim mode and passed, also call analytics.simTestPassed(score, scorePercentage).
- In handleQuit: call analytics.quizQuit(mode, answers.length, score)
- After free quiz completion (handlePracticeComplete): if user is NOT authenticated, set localStorage.setItem('permit_free_quiz_used', 'true'). Also call a new server action `markFreeQuizUsed()` that sets the cookie server-side: `cookies().set('permit_free_quiz_used', 'true', { maxAge: 365 * 24 * 60 * 60 })`. Import this from a new export in src/server/actions/auth.ts.
- After free quiz summary, show a signup prompt. This can be a state check: if no authenticated user AND quiz just completed, show a styled banner/card saying "Save your progress and track your improvement" with a Link to /login. Add this to the practice summary page (or pass a flag to the summary components).

**Add markFreeQuizUsed to src/server/actions/auth.ts:**
```typescript
export async function markFreeQuizUsed() {
  const cookieStore = await cookies()
  cookieStore.set('permit_free_quiz_used', 'true', { maxAge: 365 * 24 * 60 * 60 })
}
```
