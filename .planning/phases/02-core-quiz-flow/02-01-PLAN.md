<task type="auto" tdd="true">
  <name>Task 2: Zustand quiz store and question selection service</name>
  <files>src/stores/quiz-store.ts, src/server/db/queries/questions.ts</files>
  <behavior>
    - initSession resets state, loads questions with pre-shuffled answer options, sets mode
    - selectAnswer sets selectedAnswer and transitions answerState to "selected"
    - submitAnswer checks correctness, updates score/streak/bestStreak, records QuizAnswer, transitions to "submitted"/"revealed"
    - nextQuestion increments currentIndex, resets answer state to "idle"
    - Cannot go back: no decrementIndex or previousQuestion action
    - flagQuestion adds currentIndex to flaggedIndices set (sim mode only)
    - completeQuiz sets isComplete=true, calculates total time
    - quitQuiz sets isComplete=false (partial)
    - reset clears all state to defaults
    - Practice: selectAnswer -> submitAnswer -> revealed (auto-advance after correct, manual Next on wrong)
    - Sim: selectAnswer -> submitAnswer -> auto next (no reveal, no explanation)
    - selectPracticeQuestions fetches N random questions from seed_questions with category join
    - selectSimQuestions fetches 46 questions proportional to CATEGORY_WEIGHTS
  </behavior>
  <action>
    Create `src/stores/quiz-store.ts` implementing the full quiz state machine per Research Pattern 1 and Pattern 3:

    State fields: mode, questions (PreparedQuestion[]), totalQuestions, currentIndex, selectedAnswer, answerState, questionStartTime, answers (QuizAnswer[]), score, streak, bestStreak, flaggedIndices (Set<number>), reviewingFlagged, sessionStartTime, isComplete, isMuted.

    Actions:
    - `initSession(questions: PreparedQuestion[], mode: QuizMode)`: calls reset() first (per pitfall #4), sets questions/mode/totalQuestions, starts session timer
    - `selectAnswer(answer: string)`: only if answerState is "idle", sets selectedAnswer, transitions to "selected"
    - `submitAnswer()`: only if answerState is "selected". Compares selectedAnswer to currentQuestion.correctAnswer. Creates QuizAnswer record with timeTakenMs (Date.now() - questionStartTime). Updates score (if correct), streak (reset to 0 on wrong, increment on correct), bestStreak. For practice mode: transitions to "revealed". For sim mode: auto-advances to next question (no reveal).
    - `nextQuestion()`: increments currentIndex, resets selectedAnswer/answerState/questionStartTime. If currentIndex >= totalQuestions, triggers completeQuiz.
    - `flagQuestion()` / `unflagQuestion()`: toggles currentIndex in flaggedIndices set
    - `startFlaggedReview()`: sets reviewingFlagged=true
    - `completeQuiz()`: sets isComplete=true, calculates totalTimeMs
    - `quitQuiz()`: sets isComplete=false
    - `toggleMute()`: flips isMuted
    - `reset()`: returns all fields to initial defaults

    Computed getters (as selectors or derived):
    - `currentQuestion`: questions[currentIndex]
    - `isLastQuestion`: currentIndex === totalQuestions - 1
    - `progress`: (currentIndex + 1) / totalQuestions
    - `scorePercentage`: score / answers.length (or 0 if no answers)

    Create `src/server/db/queries/questions.ts` per Research Pattern 2:

    `selectPracticeQuestions(supabase, count=15)`: fetches count*2 questions from seed_questions with category join (select *, categories(slug, name)), shuffles array, takes first `count`. Returns questions with shuffled answer options (combine correct_answer + wrong_answers into shuffled array).

    `selectSimQuestions(supabase)`: fetches ALL seed_questions with category join. Groups by category slug. For each category, picks Math.round(46 * weight) questions from shuffled pool. Adjusts total to exactly 46 (trim extras or add from largest pool if short). Final shuffle of selected questions. Returns with shuffled answer options.

    Both functions return `PreparedQuestion[]` -- they transform the raw DB rows into PreparedQuestion shape by shuffling correct_answer into wrong_answers array.

    Helper: `shuffleArray<T>(arr: T[]): T[]` using Fisher-Yates shuffle.
  </action>
